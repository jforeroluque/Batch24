Clase 7 de febrero

Para configurar una cuenta git

(base) jorgeforero@192 DEVFBatch % git init para ubicar el archivo localmente (base)

(base) jorgeforero@192 DEVFBatch % git config -l para configurar las credenciales

(base) jorgeforero@192 DEVFBatch % git config --global user.name "jforeroluque" para configurar el username del github

(base) jorgeforero@192 DEVFBatch % git config --global user.email "j.forero@me.com" para configurar el mail del github

Para conectarla a un Git local

(base) jorgeforero@192 DEVFBatch % git status para revisar si hay commits y si hay tracked files

(base) jorgeforero@192 DEVFBatch % git add index.html Con este añado es el archivo que voy a trackear o que voy a modificar y si le doy status ya va a aparecer New;file index.html

(base) jorgeforero@192 DEVFBatch % git commit -m "Creamos el archivo base index.html" Aqui ya le pongo un mensaje al comit a lo que sea que le haya hecho

(base) jorgeforero@192 DEVFBatch % git log Con este ya entonces miramos lo ultimo que ha pasado en el archivo

Para conectarla a github (Remoto)

(base) jorgeforero@192 DEVFBatch % git remote add origin "https://github.com/jforeroluque/Batch24.git" se crea un repositorio en github y se copia esa URL, el repositorio publico

(base) jorgeforero@192 DEVFBatch % git remote -v con este se revisa que si está conectado

(base) jorgeforero@192 DEVFBatch % git push origin master Con este lo mandamos al github
